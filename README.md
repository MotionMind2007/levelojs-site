# Levelo JS Framework Website

The official website, documentation hub, and landing page for Levelo JS — a zero-runtime, component-focused JavaScript framework designed for extreme performance without the Virtual DOM.

---

## Project Purpose

This repository contains the complete source code for the official Levelo JS website. The goal of this project is to provide developers with a clear definition, comprehensive documentation, and practical examples of how Levelo JS redefines component architecture with fine-grained reactivity and scoped styling.

---

## Key Features Showcased

* **Interactive Definitions:** A clear breakdown of the Levelo JS core APIs (`state`, `effect`, `computed`, `style`).
* **Live Dashboard:** A high-performance dashboard demonstrating the real-time speed of the framework.
* **Component Playground:** Examples showcasing zero-runtime styling and declarative UI binding.

---

## Technical Stack

* **Framework:** Levelo JS (Core Reactive Runtime & Client Router)
* **Styling:** Scoped CSS Architecture
* * * **Icons:** Lucide Icons & Devicons (via CDN Integration)
* **Tooling:** Levelo CLI / Custom Bundler Config

## Getting Started

Follow these steps to run the website locally on your machine for development or contribution.

### Prerequisites

Ensure you have levelojs installed on your project.

### Installation

1. Clone the repository:
```bash
git clone https://github.com/MotionMind2007/levelojs-site
```
2. Navigate into the project directory:
```sh
cd levelojs-site
```
3. Install the dependencies:
```sh
npm install
```

---

## Development Server
To start the local development server with hot-reloading:
```sh
npm run dev
```
